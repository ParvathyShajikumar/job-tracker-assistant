# 🚀 Job Tracker Assistant

A Streamlit + SQLite app that automatically tracks job applications from email and visualizes them in an interactive dashboard.

## ✨ Features
- 📩 Reads job-related emails and stores them in a database
- 🗂️ Tracks lifecycle (Applied → Interview → Offer/Rejected)
- 📊 Interactive dashboard with filters and charts
- 💾 SQLite backend with unique keys to prevent duplicates
- 📥 Export applications as CSV

## 🛠️ Tech Stack
- Python
- Streamlit
- Plotly Express
- Pandas
- SQLite

## ⚙️ Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/job-tracker-assistant.git
   cd job-tracker-assistant

2. Create a virtual environment:
   python -m venv .venv
   source .venv/bin/activate   # On Windows: .venv\Scripts\activate

3. Install dependencies:
   pip install -r requirements.txt

4. Run the dashboard:
   streamlit run dashboard.py
